# Task-Manager-App

## Technologies used
- This project was created using HTML, CSS, React, React bootstrap, bootstrap, Typescript, and Autherization with Auth0. 
 

## Installation & setup
1. Click the link to open up my GitHub page. 

https://github.com/clounsbu/task-managaer-app

2. Once on the GitHub page click on the **green** button to the right that says **code**. This will bring a dopdown menu, you will click on **HTTPS** and copy the link below it. 
3. Open you terminal in VS code and type
```sh
git clone https://github.com/clounsbu/task-managaer-app.git
```
4. You will see a folder named **task-managaer-app**. Open the folder in **Visual Studio Code**.
5. To view the project open the terminal and type npm run dev and the app will be oploaded.


## How to use as a typical user
- Once your on the website you will see a navbar that says Task Manager App and below that text that says sign in to continue followed by a log in button.
- To access your tasks click the log in button.
- Once you click the log in button you will be brought to the Auth0 form.
- If it's your first time on the App click on the sign-up button and fill out the form. 
- If you already have an account enter your email and password and click continue. 
- After you click continue, you will be brought to  the dashboard. 
- The navbar will now include a welcome message with your email and a log out button for when you want to log out of the app. 
- Under the nav you will see the word dashboard and a button to add new task. 
- Click the add new task button to add a new task to your list.
- After clicking the button you will be brought to the add-task page where you will fill out the form. You must include the title, description, status, prioirty, and a due date for your task.  
When you are done filling out the form click the save task button and you will now see a card with all the information of your task. 
- on each card you will have 3 buttons for view details, edit, and delete.
